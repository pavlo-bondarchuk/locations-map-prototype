Map legend

Static legend for the free map provider used in this prototype.

What this prototype does

  • Works as a one-file prototype without page reloads.
  • Uses Leaflet + OpenStreetMap with no API key required.
  • Supports local search, browser geolocation, nearest locations, and URL state.
  • Single-select location behavior syncs map, list, and cards.

Marker legend

  • Blue marker: regular location.
  • Highlighted blue marker: selected location.
  • Green status: geolocation is active.

URL behavior

Search text, selected location, and current map view are saved to GET params through replaceState, so the URL can be shared without reloading the page.
